# Security Policy

## Reporting a Vulnerability

- Please email **salihcantekin@gmail.com** with the subject `RustFrame Security`.
- Include a detailed description, steps to reproduce, and possible impact.
- Do not create public issues for vulnerabilities.
- You can expect an acknowledgement within **3 business days**.

## Supported Versions

| Version | Supported |
|---------|-----------|
| dev branch | ✅
| latest release | ✅
| older releases | ⚠️ best-effort

## Preferred Fix Path

1) Coordinated disclosure via private reporting
2) Fix on `dev`, patch release cut
3) Advisory added to release notes
