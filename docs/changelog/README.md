# Changelog

This directory contains release notes for each version of RustFrame.

## Versions

| Version | Date | Highlights |
|---------|------|------------|
| [v1.1.0](v1.1.0.md) | 2025-12-28 | Multi-monitor support, custom tray icon, improved ESC behavior, real-time settings display, enhanced CI/CD |
| v0.1.0 | 2025-12-27 | Initial release with basic screen capture functionality |

## How to Use

Each version file contains:
- New features with descriptions
- Technical changes and affected files
- Dependencies added/removed
- Issues closed

## Contributing

When adding new features:
1. Document changes in the current development version file
2. Include issue/PR numbers where applicable
3. Group changes by category (Features, Fixes, Technical, etc.)
